## 0.2.0

### Minor Changes

- [#1](https://github.com/hushenv/hushenv/pull/1) [`ed7c3aa`](https://github.com/hushenv/hushenv/commit/ed7c3aab3dc2f5db94b5f0f5ffb4e5143c964657) Thanks [@lasalasa](https://github.com/lasalasa)! - Add the `mv` command (alias `rename`) to rename a secret. The value is

  `createdAt` is preserved, and `--force` overwrites an existing destination.

# Support

Thanks for using hushenv! Here's where to go depending on what you need.

- **Questions / how do I…?** — Open a
  [GitHub Discussion](https://github.com/hushenv/hushenv/discussions) (or a
  question issue if Discussions aren't enabled yet).
- **Bug reports** — Open an
  [issue](https://github.com/hushenv/hushenv/issues/new/choose) using the Bug
  Report template.
- **Feature requests** — Use the Feature Request issue template.
- **Security vulnerabilities** — **Do not** open a public issue. Follow
  [SECURITY.md](SECURITY.md) for private disclosure.

Before filing a bug, please:

1. Check existing [issues](https://github.com/hushenv/hushenv/issues).
2. Confirm you're on the latest version (`npm i -g hushenv@latest`).
3. Include your OS, Node version (`node -v`), and exact steps to reproduce.
   Never paste real secret values.

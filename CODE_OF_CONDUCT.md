# Code of Conduct

This project has adopted the [Contributor Covenant](https://www.contributor-covenant.org),
version 2.1, as its Code of Conduct. It sets the standard for a welcoming,
harassment-free community for everyone, regardless of background or identity.

Please read the full text here:

- **Contributor Covenant v2.1:** https://www.contributor-covenant.org/version/2/1/code_of_conduct/
- Plain-text version: https://www.contributor-covenant.org/version/2/1/code_of_conduct/code_of_conduct.txt

## Our pledge (summary)

We as members, contributors, and maintainers pledge to make participation in our
community a respectful and inclusive experience. We commit to acting and
interacting in ways that contribute to an open, welcoming, diverse, and healthy
community. The full pledge, standards, scope, and enforcement guidelines are
defined by the Contributor Covenant v2.1 linked above and apply in full.

## Scope

This Code of Conduct applies within all community spaces — issues, pull
requests, discussions, and any other project channel — and also when an
individual is officially representing the project in public spaces.

## Reporting

Instances of unacceptable behavior may be reported to the project maintainer at
**lasanthaslakmal@gmail.com**. All complaints will be reviewed and investigated
promptly and fairly. The maintainer is obligated to respect the privacy and
security of the reporter of any incident.

The maintainer will follow the Contributor Covenant's
[Enforcement Guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines)
when determining the consequences for any action deemed in violation of this
Code of Conduct.
